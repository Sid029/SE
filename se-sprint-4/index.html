Please choose one of the roles below:

Visitor

Citizens or visitors can use the web service to indicate whether they have entered a particular place and when they have done so

Hospital

Hospitals can use the web service to mark people as infected and track anyone else that has been in contact with an infected person.

Place

Places which are frequently visited by people can use the web service to get access to a QR code, which identifies their place. Citizens will scan this QR code to record their presence.

Agency

The agency or the evaluation client can use the web service to generate corona- related reports by collecting data from the database.
